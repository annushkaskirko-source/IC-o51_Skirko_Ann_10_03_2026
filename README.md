## Мета роботи
Реалізувати збереження та завантаження даних у програмному застосунку без використання баз даних.

## Завдання роботи
1. Реалізувати серіалізацію об’єктів у формат JSON.  
2. Реалізувати збереження даних у файл.  
3. Реалізувати завантаження даних при запуску застосунку.  
4. Інтегрувати механізм збереження з існуючою логікою програми.  

## Використані технології та підходи
- **Мова програмування:** C#  
- **Формат серіалізації:** JSON  
- **Файлова система:** Робота з файлами (`File`, `StreamReader`, `StreamWriter`)  
- **Персистентність даних:** Збереження стану об’єктів між сеансами програми  
- **Бібліотеки:** `System.Text.Json`  


## Опис реалізованого механізму серіалізації
- Серіалізація об’єктів у формат JSON через `JsonSerializer.Serialize()`.  
- Збереження даних у файл при завершенні сеансу (`.json`).  
- Завантаження даних при запуску програми через `JsonSerializer.Deserialize()`.  

## Фрагменти коду
```csharp
// Збереження даних
string json = JsonConvert.SerializeObject(data, Formatting.Indented);
File.WriteAllText(_filePath, json);

// Завантаження даних
if (!File.Exists(_filePath))
    return false;

string json = File.ReadAllText(_filePath);
var data = JsonConvert.DeserializeObject<GameData>(json);
